## Definición formal de Lista en python
Una lista es una estructura de datos ordenada mutable (que se puede modificar) que permite almacenar múltiples elementos Bajo un mismo nombre de variable.

### Características claves:
- **usa corchetes** `[]` para definir una lista.
- **tiene un orden (indices)**, lo que significa que cada elemento tiene una posición específica:
    ejemplo: 
```python
feed_instagram = ["foto_playa", "meme_de_programacion", "video_gato"]
```

En nuestro primer ejemplo: [0] es "foto_playa", [1] es "meme_de_programacion" y [2] es "video_gato", etc
